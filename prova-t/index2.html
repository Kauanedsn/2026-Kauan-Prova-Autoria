Oque eu aprendi em autoria web

Eu e minha turma aprendemos sobre html e css,incluindo as tags h...,p...,links,imagens,tags de estrutura como html,DOCTYPE,head,body e etc....

Estilos

sobre os estilos nós aprendemos sobre css interno, externo e em linha. O css em linha adicionamos as alterações que desejamos em uma unica linha. O css interno adicionamos as alterações em varias linhas e não só em uma. E o css externo criamos um novo documento para fazermos as alterações nele.
